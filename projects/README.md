# Loan Approval Project

## Project Description
This project aims to develop a loan approval system that analyzes applicant data to determine the likelihood of loan repayment.

## Installation Instructions
Follow these steps to set up the project:

1. Clone the repository:
   ```bash
   git clone https://github.com/AminKJun/AminKJun.git
   ```
2. Change to the project directory:
   ```bash
   cd AminKJun/projects
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the Loan Approval analysis, use the following command:

```bash
python loan_approval.py
```

## Dependencies
- pandas
- scikit-learn
- numpy
